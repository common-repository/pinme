a simple shortcode pin board display plugin

== Description ==

PinMe is used to display the pinterest users board or pins on there website using a simple shortcode 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pinme_pinterest` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== A brief Markdown Example ==

[pinme url=""] is the most basic url you can set 

option are 
* url="" the pinterest url you want to embed 
* type="embedBoard" ( embedPin, buttonFollow, embedUser ) are supported as well
width="300"
scale-height="340"
scale-width="120"

options with * are required
